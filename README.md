# Yunzai插件

#### 介绍
闲暇时c，v出的yunzai插件，依赖云崽实现功能。仅适配V3。

#### 插件功能

精简随机st(V3)：随机获取一张美图（可自行配置图片年龄段，仅两段）可自行配置图片类型和屏蔽群

(V3)setuplus:随机一张较为古老的美图（群聊慎用）

#### 食用方法

自行下载js文件放入Yunzai/plugins/example文件夹即可，即装即用。

精简随机st(V3)触发指令：st

(V3)setuplus触发指令：setuplus
